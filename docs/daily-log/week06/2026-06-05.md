### 学到了什么

回头看这六周，最大的收获不是记住了多少个命令或框架的 API，而是**思维方式变了**。

以前觉得"写代码"就是敲键盘，现在知道了声明式 API 的思想——K8s 里你告诉集群"我想要什么"而不是"你一步步怎么做"，这和命令式的脚本完全是两种思路。Docker 的分层缓存也让我第一次理解了"复用"不只是代码层面的，构建流程本身也可以被优化。

做 RAG 那周印象最深：LangChain 把检索、向量化、生成串在一起，看似简单的问答背后其实有一整条流水线。Agent 那周也一样，ReAct 范式本质上就是"想清楚再动手"——先观察、再推理、再行动。这些东西单独拿出来都不难，但串起来才是一个完整的能力。

### 关于坚持

连续 45 天的 commit，每天都能学到新知识，保持积极开放的思想，乐观拥抱世界的变化。**每天进步一点点，踏踏实实，脚踏实地**——回头看那个全绿的格子，比任何成就感都实在。
