### 知识点总结

| 概念 | 一句话理解 |
|------|-----------|
| messages | 不是"问一个问题"，而是"给模型完整的对话上下文" |
| System Prompt | 设定"人设"和行为规则，在 messages 最开头 |
| Function Calling | 模型当"大脑"（决定调什么），你当"手脚"（真正执行） |
| Agent | Function Calling + while 循环 = 模型自主决策 |
| ReAct | Thought → Action → Observation 循环，每步记录在 messages 中 |
| max_turns | 防止模型无限循环调工具的安全阀 |
