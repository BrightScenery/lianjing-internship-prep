## 核心认知

### 为什么 GPU 不能像 CPU 一样被 K8s 管理
- CPU/内存是通用资源，K8s 原生认识
- GPU 是特殊硬件，需要 NVIDIA Device Plugin 注册到 K8s
- GPU 最小单位是 1（不能要 0.5 张卡），不支持超卖

### 三种 GPU 共享方案
| 方案 | 隔离级别 | 适用硬件 | 核心缺点 |
|------|---------|---------|---------|
| MPS 时间片 | 无隔离 | 所有 GPU | 显存不隔离，互相影响 |
| vGPU 虚拟化 | 软件隔离 | 需要许可证 | 有性能损耗，要钱 |
| MIG 切分 | 硬件隔离 | A100/H100+ | 仅数据中心 GPU 支持 |

### GPU 排查三板斧
1. `nvidia-smi` —— 看显存、利用率、温度
2. `kubectl describe pod` —— 看 Events 错误原因
3. `kubectl exec -- nvidia-smi` —— 进容器确认 GPU 可见
