##  本周感悟

1. **Docker 的核心价值是"消除环境差异"**。联旌给客户部署大模型时，不用在每台服务器上重装 Python、配环境变量，拉个镜像就行。这就是"一次构建，到处运行"。

2. **分层是 Docker 的灵魂**。理解了分层，就理解了为什么 Dockerfile 要这样写、为什么改一行代码只重建几层、为什么 10 个应用用同一个基础镜像只占一份空间。

3. **Compose 是 K8s 的预科班**。services → Deployment, networks → Network, volumes → PVC，概念是一一对应的。单机编排理解了，分布式编排就更容易上手。

4. **容器化数据库的铁律：数据必须外置**。容器本身是短暂的，数据库/数据集必须放在 Volume 里。联旌 EaaS 平台部署 MySQL/Redis 时，底层必然是这个逻辑。
