# 每日学习日志

本目录记录每天的学习内容、产出和反思。每天一个文件，文件名格式 `YYYY-MM-DD.md`。

> 每天睡前 30 分钟，做三件事：
> 1. 写日志（复制下方模板）
> 2. 提交代码：`git add -A && git commit -m "📅 YYYY-MM-DD: 一句话总结"`
> 3. 更新 `metrics/progress.json` 中的进度数据

---

## 日志模板（每天复制以下内容，替换日期）

```markdown
# YYYY-MM-DD 学习日志

## 📌 今日目标
- [ ] 任务1
- [ ] 任务2

## ✅ 完成情况

### 学了什么
- 知识点1
- 知识点2

### 踩过的坑
- 问题 → 解决方法

### 产出
- （贴截图 / 代码片段 / 命令输出）

## 💡 今日感悟
（一句话总结今天最大的收获）

## 📊 量化数据
| 指标 | 数值 |
|------|------|
| 学习时长 | h |
| 新学命令/概念 | 个 |
| 完成练习 | 个 |
```
