### 学了什么
1. **grep**：文本搜索。`grep "ERROR" file` 筛选包含关键词的行，`-c` 统计匹配行数。
2. **管道符 `|`**：把前一个命令的输出交给后一个命令，如 `grep "ERROR" file | wc -l`。
3. **cut**：按列切割。`cut -d',' -f1` 以逗号为分隔符取第1列。
4. **sort**：排序。`sort -t',' -k3 -nr` 按第3列数字逆序排列。
5. **head / tail**：看头尾几行。`tail -n +2` 从第2行开始（跳过表头）。
6. **sed**：文本替换。`sed 's/ERROR/警告/g'` 全局替换文本。
7. **重定向 `>` / `>>`**：把命令输出写入文件，`>` 覆盖，`>>` 追加。
8. **awk**：数据处理语言。`awk -F',' '{total=$2+$3+$4}'` 可以做列计算。
9. **Shell 脚本语法**：变量赋值（`name="值"`）、条件判断（`if [ ]`）、循环（`for` / `while`）、函数定义。
