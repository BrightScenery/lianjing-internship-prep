### 学了什么
1. **ls**：列出目录内容。`ls -la` 显示隐藏文件和详细信息，`ls -lh` 带人类可读的文件大小。
2. **cd**：切换目录。`cd -` 可以回到上一个目录，`cd ..` 返回上级目录。
3. **mkdir**：创建目录。`mkdir -p a/b/c` 可以级联创建多级目录，父目录不存在时自动创建。
4. **touch**：创建空文件或更新已有文件的时间戳。
5. **cp**：复制文件。`cp file1 file2` 复制文件，`cp -r dir1 dir2` 递归复制目录。
6. **mv**：移动文件或重命名。同一个命令，目标位置不同功能就不同。
7. **cat**：查看文件内容，`>>` 追加内容，`>` 覆盖写入。
8. **rm**：删除文件。`rm -r` 删除目录，`rm -rf` 强制删除（慎用）。
9. **find**：查找文件，配合 `sort` 可以按顺序列出目录树。

### 练习内容：file-practice.sh

编写了一个 Shell 脚本，用 `mkdir`、`touch`、`echo`、`find` 等命令自动创建以下目录结构：

```
practice-demo/
├── docs/
│   └── readme.txt
├── scripts/
│   └── hello.sh
└── data/
    └── sample.csv
```

脚本代码：

```bash
#!/bin/bash
# file-practice.sh: 用命令自动创建一个目录结构

echo "开始创建 practice-demo 目录结构..."

# 创建目录
mkdir -p practice-demo/docs
mkdir -p practice-demo/scripts
mkdir -p practice-demo/data

# 创建文件并写入内容
echo "# 项目说明" > practice-demo/docs/readme.txt
echo "这是一个练习项目" >> practice-demo/docs/readme.txt

echo "#!/bin/bash" > practice-demo/scripts/hello.sh
echo 'echo "Hello from practice-demo!"' >> practice-demo/scripts/hello.sh

echo "name,age,city" > practice-demo/data/sample.csv
echo "Alice,20,Shanghai" >> practice-demo/data/sample.csv
echo "Bob,22,Beijing" >> practice-demo/data/sample.csv

echo ""
echo "目录结构创建完成！"
echo "--- tree ---"
find practice-demo -type f | sort
```

### 踩过的坑
- **问题**：一开始不理解 `>` 和 `>>` 的区别
- **原因**：`>` 会清空原文件再写入（覆盖），`>>` 是在文件末尾追加内容
- **解决**：手动用两种方式试了一下，发现 `>` 之后再 cat 只有新内容，`>>` 会累积内容
