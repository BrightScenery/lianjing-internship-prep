### 学了什么
    1.今天我了解了 WSL2 是什么，WSL2（Windows Subsystem for Linux 2）是微软在 Windows 中提供的一项功能，允许用户直接在 Windows 上运行原生的 Linux 内核和发行版（如 Ubuntu），无需安装虚拟机或双系统，让我在 Windows 里能像运行普通程序一样跑 Linux 命令和应用。

    2.已成功安装并进入 Ubuntu 终端，能够通过 /mnt/c/ 访问 Windows 文件系统。

    3.运行第一个 Python 脚本,
    使用 cd 命令切换到脚本所在目录：
    cd /mnt/c/Users/.../hello-shell/

    使用 python3 hello.py 执行 Python 脚本，成功跑通。

    4.终端基本操作
    路径切换：cd 进入目录，理解 WSL 中 Windows 路径的挂载点（/mnt/c/ 对应 C 盘）。

    运行程序：直接输入命令名（如 python3）加参数。

    命令组合：可以用 && 连接多条命令，例如 cd 目录 && python3 脚本.py。


### 踩过的坑
    WSL 中无法访问 Windows 桌面文件
    解决：使用 /mnt/c/Users/你的用户名/Desktop 路径，
    注意 WSL 下路径区分大小写。
