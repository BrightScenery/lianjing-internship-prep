### 学了什么
1. **WSL2 是什么**：WSL2（Windows Subsystem for Linux 2）是微软在 Windows 中提供的功能，允许直接在 Windows 上运行原生 Linux 内核和发行版（如 Ubuntu），无需虚拟机或双系统。可以在 Windows 里像运行普通程序一样跑 Linux 命令和应用。

2. **安装 Ubuntu 环境**：成功安装并进入 Ubuntu 终端，能够通过 `/mnt/c/` 访问 Windows 文件系统。

3. **运行第一个 Python 脚本**：
   - 使用 `cd` 命令切换到脚本所在目录：`cd /mnt/c/Users/H1831/Desktop/lianjing-internship-prep/projects/hello-shell/`
   - 使用 `python3 hello.py` 执行脚本，成功输出结果

4. **终端基本操作**：
   - 路径切换：`cd` 进入目录，理解 WSL 中 Windows 路径的挂载点（`/mnt/c/` 对应 C 盘）
   - 运行程序：直接输入命令名（如 `python3`）加参数
   - 命令组合：用 `&&` 连接多条命令，例如 `cd 目录 && python3 脚本.py`

### 踩过的坑
- **问题**：WSL 中直接用 `cd Desktop` 找不到 Windows 桌面文件
- **原因**：WSL 下路径区分大小写，且 Windows 磁盘挂载在 `/mnt/c/` 下
- **解决**：使用完整路径 `/mnt/c/Users/H1831/Desktop`
